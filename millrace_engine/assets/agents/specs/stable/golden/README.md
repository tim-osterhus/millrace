# Golden Stable Specs

This folder stores immutable golden specs generated by Mode A:

- Path contract: `agents/specs/stable/golden/<spec_id>__<slug>.md`
- Source template: `agents/specs/templates/golden_spec_template.md`
- Required sections include `Req-ID` traceability, `Assumptions` ledger, `Structured Decision Log`, and `Interrogation` record.
- Frontmatter should include `golden_version` and `base_goal_sha256` for version traceability.

Golden specs are the canonical parent artifact for downstream `PHASE_` specs.
